# Introduction

This page hosts documentations and specifications for some of the cryptographic
algorithms of [Mina](https://minaprotocol.com/).
[For the Rust documentation, see here](https://o1-labs.github.io/proof-systems/rustdoc/).

Note that this book is a work in progress, not necessarily reflecting the
current state of Mina.
