fn chunk_polynomial(&self, zeta_n: F, n: usize) -> Self

Multiplies the chunks of a polynomial with powers of zeta^n to make it of degree n-1. For example, if a polynomial can be written f = f0 + x^n f1 + x^2n f2 (where f0, f1, f2 are of degree n-1), then this function returns the new semi-evaluated f'(x) = f0(x) + zeta^n f1(x) + zeta^2n f2(x).
