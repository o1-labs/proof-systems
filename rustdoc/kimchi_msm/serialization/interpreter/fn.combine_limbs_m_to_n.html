Helper function for limb recombination.

Combines an array of M elements (think N_LIMBS_SMALL) into an array of N elements (think N_LIMBS_LARGE) elements by taking chunks a_i of size K = BITSIZE_N / BITSIZE_M from the first, and recombining them as a_i * 2^{i * 2^LIMB_BITSIZE_SMALL}.
