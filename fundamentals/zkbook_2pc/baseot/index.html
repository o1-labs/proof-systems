Base OT

We introduce the CO15 OT as the base OT.

Notation. The protocol is described over an additive group (G,G,p,+)(\bG,\G,p,+)(G,G,p,+) of prime order ppp generated by G\GG. Denote H:(G×G)×G→{0,1}λ\sH:(\bG\times \bG)\times\bG\rightarrow \bit^\lambdaH:(G×G)×G→{0,1}λ as a key-derivation function to extract a λ\lambdaλ-bit key from group elements.

Note that H\sHH here is different from the tweakable hash function in garbled circuit.

  • The sender sends S\SS to the receiver, who aborts if S∉G\S\notin\bGS∈/G.

  • For i∈[n]i\in[n]i∈[n], the receiver samples xi←Fpx_i\leftarrow \bF_pxi​←Fp​, and computes: Ri=biS+xiG\R_i = b_i\S+x_i\GRi​=bi​S+xi​G
